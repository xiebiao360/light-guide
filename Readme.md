## 项目说明

Light Guide 是一款用于监控和管理服务器环境的轻量级工具。它使用 Rust 后端和 Vue3 前端构建，打包成单一二进制文件，便于在客户环境中部署和使用。

### 功能一：仪表盘

仪表盘提供服务器环境的实时监控，包括：

- **系统基本信息**：
  - 操作系统类型和版本
  - CPU架构
  - 主机名和内核版本
  - 运行时间(uptime)

- **资源使用情况**：
  - CPU：整体使用率、每核心使用率、温度(如可获取)
  - 内存：总量、已用量、可用量、缓存大小
  - 磁盘：各分区使用率、IO读写速率
  - 网络：入站/出站流量、连接数

- **负载指标**：
  - 系统负载均值(1分钟/5分钟/15分钟)
  - 运行中的进程数
  - 用户会话数

通过WebSocket实现实时数据更新，提供直观的图表展示。

### 功能二：日志管理

提供对指定工作目录中日志文件的管理和分析：

- **文件浏览**：浏览工作目录中的文件和子目录
- **日志查看**：支持实时滚动输出显示日志内容
- **全文检索**：使用tantivy实现简单关键词搜索功能

### 安装与使用

1. **下载二进制文件**：从发布页面下载适合您系统的二进制文件

2. **运行服务**：
   ```bash
   ./light-guide --work-dir /path/to/logs
   ```

3. **访问仪表盘**：
   在浏览器中访问 `http://localhost:3000`

### 配置选项

| 参数 | 说明 | 默认值 |
|------|------|--------|
| --port | 监听端口 | 3000 |
| --work-dir | 工作目录路径 | 当前目录 |
| --db-path | 数据库文件路径 | guide.db |

### 技术架构

- **后端**：Rust + Axum + SQLite + tantivy
- **前端**：Vue3 + 图表库
- **通信**：WebSocket + REST API

### 注意事项

- 工具需要对指定的工作目录有完全访问权限
- 适合单服务器环境监控，不支持多服务器集中管理
- 监控数据不会长期保存，仅供实时
